## What is reactify?

reactify is a React framework built on Fastify. It gives you server-side rendering, React Server Components, server actions, and file-system routing out of the box.

## Getting Started

Create a new project and start the development server to see reactify in action. The framework handles all the complex wiring so you can focus on building your application.

## Key Features

- **File-system routing** — pages in the pages directory become routes automatically
- **RSC** — async server components that stream to the client for fast page loads
- **Server Actions** — call server functions directly from client components
- **SSR Streaming** — Suspense boundaries stream HTML progressively for improved perceived performance
